
y(t)  = a*t^3 + b*t^2 + c*t + d
y'(t) = 3*a*t^2 + 2*b*t + c

--- solve

y(0) = d
y(1) = a+b+c+d

y'(0) = c
y'(1) = 3*a + 2*b + c

y(1)  = a+b+y'(0)+y(0) =>
a = y(1)-y'(0)-y(0)-b

y'(1) = 3*a + 2*b + y'(0)
y'(1) = 3*y(1)-3*y'(0)-3*y(0)-3*b + 2*b + y'(0)
y'(1) =  -3*y(0) + 3*y(1) - 2*y'(0) -b
b = -3*y(0) + 3*y(1) - 2*y'(0) - y'(1)

a = y(1)-y'(0)-y(0)+3*y(0) - 3*y(1) + 2*y'(0) + y'(1)

--- result

a = 2*y(0) - 2*y(1) + y'(0) + y'(1)
b = -3*y(0) + 3*y(1) - 2*y'(0) - y'(1)
c = y'(0) 
d = y(0)
